# HandSub

빠르고 가벼운 데스크톱 메모 앱. 단축키 한 번으로 언제든 메모하세요.

![HandSub Screenshot](https://handsub.com/screenshot.png)

## 다운로드

| OS | 다운로드 |
|----|----------|
| **macOS** | [HandSub.dmg](https://app.handsub.com/handsub-mac.dmg) |
| **Windows** | [HandSub.exe](https://app.handsub.com/handsub-win.exe) |

---

## 설치 방법

### macOS

1. DMG 파일 다운로드
2. 파일 열고 `HandSub.app`을 Applications 폴더로 드래그
3. 앱 실행

> **"손상되었기 때문에 열 수 없습니다" 메시지가 뜨면:**
>
> 터미널을 열고 아래 명령어를 실행하세요:
> ```bash
> xattr -cr /Applications/HandSub.app
> ```

### Windows

1. EXE 파일 다운로드
2. 설치 파일 실행
3. 설치 완료 후 앱 실행

## 주요 기능

### 글로벌 단축키
어떤 앱을 사용 중이든 단축키로 즉시 메모창 열기
- **메모 열기**: `Cmd+Shift+Space` (Mac) / `Ctrl+Shift+Space` (Win)
- **새 메모**: `Cmd+Shift+N` (Mac) / `Ctrl+Shift+N` (Win)

### 자동 저장
입력과 동시에 자동 저장. 저장 버튼 필요 없음.

### 메모 전달
이메일 주소만으로 다른 HandSub 사용자에게 메모 전달

### 스니펫 (단축어)
자주 쓰는 텍스트나 Webhook 연동을 단축어로 등록

### 미디어 지원
이미지, 동영상 복사/붙여넣기 또는 드래그&드롭

### 링크 미리보기
URL 입력 시 자동으로 미리보기 생성 (YouTube 지원)

### 체크리스트
`[ ]` 입력으로 체크박스 자동 생성
